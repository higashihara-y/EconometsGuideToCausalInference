1.1 シミュレーションデータの生成

1.1.1 定数とパラメータの設定

Code 
set.seed(1)
N <- 1000           #個体数
N_1 <- 500　　　　　#処置群のサイズ
G <- c("nc", "co")　#遵守タイプ
bias <- 0.5　　　　 #除外制約が満たされない場合のバイアス
tau_population <- c(1, 2) #母集団における平均処置効果

1.1.2 データの生成

Code 
outcome_potential <- 
  1:length(G) %>% 
  purrr::map(
    function(n) {
      df <- tibble::tibble(
        g = G[n],
        y_0 = rnorm(N / length(G),
                    mean = 0,
                    sd = 1),
        y_1 = rnorm(N / length(G),
                    mean = tau_population[n],
                    sd = 1),
        y_z = rnorm(N / length(G),
                    mean = bias,
                    sd = 1)
      )
      return(df)
    }
  ) %>% 
  dplyr::bind_rows()

tau <- 
  outcome_potential %>% 
  dplyr::group_by(g) %>% 
  dplyr::summarise(
    tau = mean(y_1 - y_0)
  ) %>% 
  dplyr::ungroup()

    y = mean(y)

1.2.5 \(\tilde{\tau}_{Y,co} = \tau_{late}\)の手動計算

結果の処置割当効果を処置受取の処置割当効果で割ることによって局所平均処置効果の推定値（＝遵守者の平均処置効果）を算出。

Code 
tau_late_excluded <- 
  tau_tilde_y_excluded /
  tau_tilde_d_excluded

tau_late_excluded
[1] 2.083201

母集団におけるパラメータ(= 2)を適切に推定している。

Code 
tau_late_included <- 
  tau_tilde_y_included /
  tau_tilde_d_included

tau_late_included
[1] 2.60888

除外制約が満たされないことで、母集団におけるパラメータ(= 2)をバイアス分過大評価している。
