1.2.2 rdplotによるプロット

デフォルトのbinselect = "esmv"は、生データの分散をビン平均値の分散で近似し、分散が等間隔になるように分割する方法。

Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s
)

ビンの区切り方はbinselectオプションを指定。 esまたはqsはいずれもサポート全体の平均二乗誤差を最小化するようにビン数を選ぶ方法。

esは等間隔でビンを区切るため、極端に少ない観測の平均をその他の平均と同等であるかのように見せる恐れがある。qsのビン選択では、各ビンに同じ観測数が入る。

Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s,
  binselect = "es"
)
Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s,
  binselect = "qs"
)

2.2.2 バンド幅とバイアス・分散の関係

パイロットバンド幅（=バイアス推定に用いるバンド幅 “b”）を固定し、推定バンド幅をMSE最適なものから変化させることで、バイアスと分散が変化する。

Code 
result_opt <- 
  rdrobust::rdrobust(
    y = dgp_lee$y,
    x = dgp_lee$s
  )

table <- 
  data.frame(
    case = "optimal",
    absolute_bias = sum(abs(result_opt$bias)),
    se_raw = result_opt$se[1],
    bandwidth_h = result_opt$bws[1,1],
    bandwidth_b = result_opt$bws[2,1]
  )

result_small <- 
  rdrobust::rdrobust(
    y = dgp_lee$y,
    x = dgp_lee$s,
    h = result_opt$bws[1,1] / 10,
    b = result_opt$bws[2,1]
  )

table <- 
  rbind(table,
        data.frame(
          case = "small",
          absolute_bias = sum(abs(result_small$bias)),
          se_raw = result_small$se[1],
          bandwidth_h = result_small$bws[1,1],
          bandwidth_b = result_small$bws[2,1]
        ))

result_large <- 
  rdrobust::rdrobust(
    y = dgp_lee$y,
    x = dgp_lee$s,
    h = result_opt$bws[1,1] * 4,
    b = result_opt$bws[2,1]
  )

table <- 
  rbind(table,
        data.frame(
          case = "large",
          absolute_bias = sum(abs(result_large$bias)),
          se_raw = result_large$se[1],
          bandwidth_h = result_large$bws[1,1],
          bandwidth_b = result_large$bws[2,1]
        ))
