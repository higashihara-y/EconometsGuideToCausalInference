1.2.1 rdrobustパッケージによる推定

  rdrobust::rdrobust(
    y = dgp_lee$y,
    x = dgp_lee$s
  )

1.2.2 rdplotによるプロット

デフォルトのbinselect = "esmv"は、生データの分散をビン平均値の分散で近似し、分散が等間隔になるように分割する方法。

Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s
)

ビンの区切り方はbinselectオプションを指定。 esまたはqsはいずれもサポート全体の平均二乗誤差を最小化するようにビン数を選ぶ方法。

esは等間隔でビンを区切るため、極端に少ない観測の平均をその他の平均と同等であるかのように見せる恐れがある。qsのビン選択では、各ビンに同じ観測数が入る。

Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s,
  binselect = "es"
)
Code 
rdrobust::rdplot(
  y = dgp_lee$y,
  x = dgp_lee$s,
  binselect = "qs"
)
